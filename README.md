## OsmAnd

**OsmAnd** is an offline maps and navigation application for Android, iOS, and web. It is designed for people who need reliable maps without depending on a constant internet connection.

OsmAnd is useful for:

- Offline navigation
- Driving, cycling, walking, hiking, and traveling
- GPX tracks and route planning
- Topographic maps, contour lines, and terrain data
- OpenStreetMap-based search and map data
- Privacy-focused navigation without mandatory online tracking

Useful links:

- Website: [osmand.net](https://osmand.net/)

- GitHub organization: [github.com/osmandapp](https://github.com/osmandapp)
