What this tool does

OsmAnd GPX favorites and waypoint files can be imported into other map tools, but external tools may use only a default icon. This converter tries to preserve useful waypoint details by mapping OsmAnd icon names to visually similar Google-style icons.

  1. Export or prepare a GPX file with OsmAnd waypoints.
  2. Drop the file onto this page, or use the file picker.
  3. Download the generated KML file.
  4. Import the KML into Google My Maps or another KML-compatible tool.
This browser version converts waypoint data only. GPX track geometry is not converted here. Unknown OsmAnd icons are marked with an x-cross style so they are easy to find and improve later.
