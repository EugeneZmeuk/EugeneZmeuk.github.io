Google Map to OsmAnd GPX Package

Build a ZIP package of OsmAnd-style GPX files from a Google My Maps KML/KMZ export.

What it does

Create an OsmAnd GPX package from a Google My Maps export. Tracks are saved as separate GPX files, and waypoints are grouped into a waypoint GPX file.

  1. Paste a Google My Maps ID to open the KML export link, or export KML/KMZ manually.
  2. Drop the exported KML/KMZ file here.
  3. Download a ZIP package and copy/import the GPX files into OsmAnd.
This static browser page cannot silently fetch private Google maps. The map must be shared/exportable, then uploaded here as KML/KMZ.
