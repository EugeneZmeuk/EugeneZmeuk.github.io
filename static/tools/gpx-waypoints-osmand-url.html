OsmAnd helper tool

GPX Waypoints to OsmAnd URL

Convert GPX waypoints or route points into an OsmAnd Web Map route link. Upload a file, choose a routing profile, and open the generated route directly in the browser.

What it does

The tool reads route points from a GPX file and builds an OsmAnd Web Map URL with a start point, optional via points, an end point, and the selected routing profile.

Supported data

  • GPX files with at least two wpt points
  • GPX files with rtept route points
  • Car, bicycle, pedestrian, and motorcycle profiles

How to use

  • Drag and drop a GPX file into the upload area, or choose it manually.
  • Select the routing profile you want to use in OsmAnd Web Map.
  • Open the generated link to check the route on OsmAnd Web Map.
