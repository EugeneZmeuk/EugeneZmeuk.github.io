# My OsmAnd Tools

Small helper tools I use for OsmAnd-related testing, support, documentation, and product workflows.

## OsmAnd GPX to KML Converter

Convert OsmAnd GPX waypoints into a KML file for Google My Maps, with similar icon styles and colors where possible.

<a href="https://eugenezmeuk.github.io/tools/osmand-gpx-to-kml.html" target="_blank" rel="noreferrer">Open tool</a>

## GPX Waypoints to OsmAnd URL

Upload a GPX file with waypoints or route points, choose a routing profile, and open the generated route directly in OsmAnd Web Map.

<a href="https://eugenezmeuk.github.io/tools/gpx-waypoints-osmand-url.html" target="_blank" rel="noreferrer">Open tool</a>

## Google Map to OsmAnd GPX Package

Create a ZIP package of OsmAnd-style GPX files from a Google My Maps KML or KMZ export, with tracks, waypoints, and layers.

<a href="https://eugenezmeuk.github.io/tools/google-map-to-osmand-gpx.html" target="_blank" rel="noreferrer">Open tool</a>

## KML to OsmAnd GPX Converter

Convert a Google My Maps KML or KMZ export into a single OsmAnd-style GPX file with tracks, waypoints, colors, and icons.

<a href="https://eugenezmeuk.github.io/tools/kml-to-osmand-gpx.html" target="_blank" rel="noreferrer">Open tool</a>

## Google Maps Icons to OsmAnd GPX

Convert Google My Maps placemarks into OsmAnd GPX waypoints with similar OsmAnd icons where possible.

<a href="https://eugenezmeuk.github.io/tools/gmap-icons-to-osmand.html" target="_blank" rel="noreferrer">Open tool</a>
